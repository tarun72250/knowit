What is Dot Net

.NET is an open source developer platform, created by Microsoft, for building many different types of applications. .NET is a free, cross-platform, open source developer platform for building many different types of applications. With .NET, you can use multiple languages, editors, and libraries to build for web, mobile, desktop, games, IoT, and more.


Languages Basics

You can write .NET apps in C#, F#, or Visual Basic.

  • C# is a simple, modern, object-oriented, and type-safe programming language.
  • F# is a programming language that makes it easy to write succinct, robust, and performant code.
  • Visual Basic is an approachable language with a simple syntax for building type-safe, object-oriented apps.
  • C# (pronounced "C sharp") is a simple, modern, object-oriented, and type-safe programming language.
  • Its roots in the C family of languages makes C# immediately familiar to C, C++, Java, and JavaScript programmers.
